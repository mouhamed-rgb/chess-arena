# ♟ Chess Arena — شطرنج أونلاين Real-time

لعبة شطرنج حقيقية لشخصين عبر الإنترنت، مبنية بـ Node.js + Socket.io + chess.js

---

## 🗂 هيكل المشروع

```
chess-online/
├── server.js           ← السيرفر (Node.js + Express + Socket.io)
├── package.json
└── public/
    ├── index.html      ← هيكل الواجهة
    ├── style.css       ← التصميم Dark Cinematic
    └── client.js       ← منطق اللعبة (Chess.js + Socket.io)
```

---

## 🚀 تشغيل المشروع محلياً

### المتطلبات
- Node.js v16 أو أحدث
- npm

### الخطوات

```bash
# 1. فك الضغط وادخل المجلد
cd chess-online

# 2. تثبيت التبعيات
npm install

# 3. تشغيل السيرفر
npm start
# أو مباشرة:
node server.js

# 4. افتح المتصفح على:
http://localhost:3000
```

---

## 🎮 طريقة اللعب

### اللاعب الأول (المضيف):
1. اضغط **"إنشاء غرفة جديدة"**
2. ستحصل على **رمز الغرفة** (6 أحرف) ورابط مباشر
3. أرسل الرابط أو الرمز لصديقك
4. انتظر حتى يدخل — تبدأ المباراة تلقائياً
5. أنت **اللاعب الأبيض** وتبدأ أولاً

### اللاعب الثاني (الضيف):
1. افتح الرابط مباشرة، أو اضغط **"انضمام لغرفة"** وأدخل الرمز
2. أنت **اللاعب الأسود**

## 🌐 نشر على الإنترنت

### Railway / Render / Heroku:
```bash
# أضف متغير البيئة:
PORT=3000

# ملف Procfile (لـ Heroku):
echo "web: node server.js" > Procfile
```

### نشر محلي على الشبكة:
```bash
# شغّل السيرفر
node server.js

# اعرف IP جهازك
ipconfig   # Windows
ifconfig   # Mac/Linux

# شارك الرابط:
http://192.168.x.x:3000
```
